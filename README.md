# Lista de Tarefas – Exercício JavaScript

Aplicação de lista de tarefas desenvolvida como exercício de manipulação do DOM com JavaScript puro.

## Funcionalidades

- Adicionar tarefas com botão ou tecla Enter
- Marcar tarefa como concluída (texto riscado)
- Remover tarefas com o botão ✕
- Validação de campo vazio com mensagem de erro no DOM
- Persistência com `localStorage` (tarefas salvas ao recarregar a página)
- Filtro por status: Todas / Pendentes / Concluídas

## Tecnologias

- HTML5
- CSS3
- JavaScript puro (sem frameworks ou bibliotecas)

## Como executar

Abra o arquivo `lista-de-tarefas.html` diretamente no navegador. Nenhuma instalação necessária.

## Conceitos praticados

- Manipulação do DOM com `createElement` e `appendChild`
- Alternância de classes com `classList.toggle`
- Remoção de elementos com `remove()`
- Captura de eventos com `addEventListener`
- Persistência de dados com `localStorage`
